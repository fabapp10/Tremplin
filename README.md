# Tremplin

Assistant de candidature propulsé par l'IA : génération de CV, lettres de motivation ciblées, préparation d'entretien et analyse de photo de CV.

Ce dépôt contient le prototype de l'interface (React) et le guide de mise en production.

## Contenu

- **`tremplin.jsx`** — le prototype complet de l'application (landing page, connexion + abonnements simulés, et les 4 outils : CV, lettre, entretien, photo). Composant React autonome.
- **`guide-mise-en-ligne-tremplin.md`** — guide pas à pas pour transformer le prototype en site réel en ligne (hébergement, connexion Google, paiement Stripe, base de données, clé API côté serveur, domaine, légal).

## Aperçu rapide du prototype

`tremplin.jsx` est un composant React unique. Pour le lancer dans un vrai projet, voir le guide (Phase 1) : création d'un projet Next.js, puis import du composant.

## Statut

Prototype de démonstration. La connexion Google et le paiement sont simulés ; voir le guide pour les rendre réels.
